# Digital Notary Browser Extension

Minimal browser extension for quick access to Digital Notary DApp.

## Quick Start
1. Download from browser store
2. Click the Digital Notary icon in toolbar
3. Press "Open Application"
4. Connect MetaMask wallet

## Development
### Load Unpacked Extension
1. Clone this repository
2. Open `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the `extension` folder

## Files
- `manifest.json` - Extension configuration (Manifest V3)
- `popup.html` - Popup user interface
- `popup.js` - Popup functionality
- `icons/` - Extension icons (16x16, 48x48, 128x128)

## Notes
- Zero permissions required
- Opens PWA version for Microsoft Store
- Main website: https://digitalnotary.online
